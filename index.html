Mastering Meteor Class

We will learn together and guide you step-by-step to show you the path to Meteor mastery. Skip the steep learning curve, within two days extensive session you'll be able to build real-time and well crafted application.

When I started use Meteor, I didn't know how to properly organize my files or how to deploy Meteor app. Don't you worry, I've been through it and I can guide you.

You will learn how to:

  • file organization
  • create database collections
  • publication and subscription
  • login and registration system
  • deploying to the server
  • and much more!

My main goal isn't just to teach you how to build one Meteor app. I want to give you the tools and know-how to build dozens of apps. 10% theory and 90% practical stuff!
